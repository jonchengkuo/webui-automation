This directory contains a Selenium-based UI automation framework and examples.

Setup and Run
=============

1. Install Firefox (e.g., 47.0.1), Java SDK 8, Maven (3.x), and Eclipse.
   You also need to configure the Maven proxy settings if you are behind
   a firewall.

2. Open Eclipse and import this project:
   1) Select File > Import... > General > Existing Projects into Workspace.
   2) At "Select root directory", click "Browse" and choose the directory
      path at webui-automation/java-framework.
   3) Select the "WebUI-Automation-Java-Framework" project and click "Finish".

3. To run an example, right-click the example program and then select
   Run As > Java Application.
